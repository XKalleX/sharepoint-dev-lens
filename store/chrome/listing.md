# Chrome Web Store Listing – SharePoint Dev Lens

## Name
SharePoint Dev Lens

## Short Description (max 132 chars)
Inspect SharePoint lists, GUIDs, fields, permissions and developer URLs directly in Chrome.

## Description
SharePoint Dev Lens is a browser extension for SharePoint developers, consultants and Microsoft 365 administrators.

It helps you inspect SharePoint sites directly in Chrome:

📋 LISTS & LIBRARIES
• View all lists and document libraries with their GUIDs
• See BaseTemplate, item count, versioning and approval status
• Identify lists with unique permissions or large item counts
• Copy GUIDs, REST URLs and paths with one click

🔍 FIELD INSPECTOR
• InternalName, StaticName, Field GUID for every field
• Type, required, indexed, read-only status at a glance
• Lookup targets, choice values, default values
• Generate REST, PnPjs and Power Automate snippets per field

🏠 SITE INFORMATION
• Web ID, site URL, template, language (LCID), Hub Site status
• SharePoint group details (Owners, Members)
• Direct links to Site Settings, Site Contents, Permissions, Recycle Bin

🔗 DEVELOPER QUICK LINKS
• Instant access to /_layouts/15/ admin pages
• REST API endpoints for the current site
• Custom URL opener for quick navigation

⬇️ EXPORT
• Export site data as JSON, Markdown or CSV
• Perfect for documentation, handovers and client reports

⚠️ GOVERNANCE WARNINGS
• Lists exceeding the 5,000-item view threshold
• Non-indexed lookup columns on large lists
• Versioning disabled, unique permissions detected
• Hidden lists

🔒 PRIVACY
• No data collection, no external servers
• Works entirely within your browser using your authenticated SharePoint session
• No OAuth flow, no token storage, no telemetry
• "This extension does not collect, sell or transmit SharePoint content."
